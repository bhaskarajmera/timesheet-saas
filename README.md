# Timesheet SaaS

A simple SaaS-style Timesheet Management App built using Next.js.

## Tech Stack

- Next.js (src/App Router)
- NextAuth.js (Authentication)
- TailwindCSS (Styling)
- Axios (API calls)

## Features

- Split screen login page
- Dummy authentication using NextAuth
- Session based token storage
- Dashboard with timesheet table
- Internal API routes
- Modular reusable components
- Responsive layout
- End Session on Signout

## Steps for Setup

1. Clone repo
   git clone repo-url

2. Install dependencies
   npm install

3. Run dev server
   npm run dev

4. Open
   http://localhost:3000
